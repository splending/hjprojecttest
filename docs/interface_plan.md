# 高校统一门户产品界面规划

## 信息架构
- **全局导航**：底部 Tab Bar 划分为“首页”“课程”“资讯”“服务”“我的”。
- **横向扩展模块**：在服务 Tab 内提供常用办事入口，使用卡片化组件便于扩展。
- **通知体系**：首页展示横幅通知和任务列表，资讯页面提供分类筛选。

## 核心界面
1. **home.html — 首页仪表盘**
   - 今日课表、待办任务、常用服务快捷入口。
   - 顶部欢迎 Banner，使用校园图像强化氛围。
2. **news.html — 校园资讯**
   - 新闻轮播、公告分类、热门活动卡片。
3. **courses.html — 我的课程**
   - 周视图课表、课程资料快捷访问、教师联系人。
4. **schedule.html — 日程与考试**
   - 日历控件、考试安排、提醒设置。
5. **library.html — 图书与学习资源**
   - 图书搜索、推荐书单、学习空间预约。
6. **grades.html — 成绩与学业进度**
   - GPA 概览、成绩列表、学分进度条。
7. **map.html — 校园地图与导航**
   - 交互地图截屏、热门地点列表、路线建议。
8. **services.html — 校园服务大厅**
   - 维修报修、宿舍服务、后勤工单进度卡片。
9. **profile.html — 个人中心**
   - 数字校园卡、认证信息、快捷入口。
10. **settings.html — 设置与通知**
    - 账号安全、消息提醒、主题切换。

## 交互逻辑
- 底部导航在所有页面保持一致，通过高亮指示当前模块。
- 卡片式组件提供统一交互：点击进入详情、右上角图标提供快捷操作。
- 服务申请表单采用分步式，保证信息填写清晰。
- 通知消息支持左右滑动标记已读（原型中以操作提示体现）。

## 视觉与响应式策略
- **主题**：蓝白主色 + 品牌高亮色（#2563EB）。
- **字体**：使用系统字体 `SF Pro`、`Inter`、`PingFang SC` 组合。
- **图片**：采用 Unsplash 高校场景、学习场景图片增强真实感。
- **布局**：移动端采用 393×852 模拟 iPhone 15 Pro，PC 端在 index.html 将多个界面平铺展示以利评审。
