新知识采集系统（静态 HTML 原型）

面向学校多模态文档大模型建设，提供统一接入、自动解析、全流程监控与可追溯的数据采集能力。

1.1.1 整体能力升级

  • 支持结合学校前期建设的多模态文档大模型能力。
  • 形成新一代知识采集系统，统一管理多源数据采集流程。

1.1.2 统一接入能力

  • 支持结构化、半结构化、非结构化数据统一接入。
  • 支持 Word/PDF/Excel/PPT/TXT/图片/音视频/扫描件。
  • 采集方式支持 API 接口、数据库直连、批量文件导入。

1.1.3 多模态解析能力 OCR + ASR + 视频解析

  • 集成高精度 OCR，支持扫描文档与图片文字精准提取。
  • 支持表格结构还原与版面分析。
  • 集成语音识别与视频解析，实现自动转写与内容提取。

1.1.4 ★ 私有化部署与并行处理

  • 支持私有化部署的多模态大模型解析能力。
  • 支持多进程并行处理，提升吞吐效率。
  • 确保数据不出校、不依赖外部公网服务。

1.1.5 文件处理自动化

  • 支持批量文件快速上传。
  • 支持自动清洗、去重与格式标准化处理。

1.1.6 结构化知识抽取

  • 从非结构化文档中自动抽取实体、关系、属性等知识信息。
  • 为后续知识库入库和检索提供结构化基础。

1.1.7 多元信息完整提取

  • 完整提取图片、表格、公式、附件、超链接等信息。
  • 保留多模态内容完整性，避免关键上下文丢失。

1.1.8 目录结构识别

  • 自动识别章节层级与目录结构。
  • 生成可浏览的目录树，便于后续检索与阅读。

1.1.9 采集监控与审计

  • 全流程监控任务状态。
  • 提供成功率、数据量统计与异常告警。
  • 支持日志审计，确保数据来源可追溯。

统一格式转换与备份策略

系统将所有上传文档统一转换为便于管理的标准格式（如 Markdown、TXT），同时保留原始文件备份。
